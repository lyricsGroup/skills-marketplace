# skills-marketplace

这个插件提供 6 个职责清晰的工程技能。

## 目录约定

- `skills/code-analysis/`: 分析代码并输出结论
- `skills/code-planning/`: 制定或修改方案并产出 Markdown
- `skills/senior-frontend/`: 资深前端实现
- `skills/senior-backend/`: 资深后端实现
- `skills/code-ui-test/`: 测试并检查 UI
- `skills/code-review/`: 代码审核并输出结论
- `hooks/hooks.json`: 预留 hooks 配置
- `scripts/validate-skills.sh`: skill 结构校验
